# poplin

The goal of poplin package is to provide simple tools for processing
LC/MS metabolomics data processing, including

  - Missing value imputation
  - Data normalization
  - Dimension reduction
  - Visualization

To install this package,

``` r
remotes::install_github("jaehyunjoo/poplin")
```
